Apuntes – Lenguajes de Marcas (HTML)

Plantillas listas para copiar y pegar · Unidad Didáctica 2

1) Estructura básica de un documento

Usa lang="es", codificación UTF‑8 y la meta viewport para responsive.
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Descripción de la página">
    <title>Título de la página</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <header>
      <h1>Título principal</h1>
      <nav>
        <ul>
          <li><a href="index.html">Inicio</a></li>
          <li><a href="about.html">Sobre nosotros</a></li>
          <li><a href="contact.html">Contacto</a></li>
        </ul>
      </nav>
    </header>

    <main>
      <section>
        <h2>Subtítulo o sección</h2>
        <p>Texto o contenido de ejemplo.</p>
      </section>
    </main>

    <footer>
      <p>&copy; 2025 Mi página web.</p>
    </footer>
  </body>
</html>

2) Etiquetas básicas de texto

Encabezados jerárquicos, párrafos y formatos de énfasis.
<h1>Encabezado principal</h1>
<h2>Subtítulo</h2>
<h3>Sección</h3>

<p>Esto es un párrafo.</p>

<b>Negrita (visual)</b>
<strong>Importante (semántico)</strong>
<i>Cursiva (visual)</i>
<em>Énfasis (semántico)</em>

<blockquote>
  «Cita larga o destacada.»
</blockquote>

<hr> <!-- Línea horizontal -->
<br> <!-- Salto de línea -->

3) Enlaces

Usa textos descriptivos, evita «haz clic aquí».
<!-- Enlace externo -->
<a href="https://www.ejemplo.com">Visita mi sitio</a>

<!-- Enlace interno (ancla) -->
<a href="#seccion1">Ir a la Sección 1</a>
<h2 id="seccion1">Sección 1</h2>

<!-- Enlace de correo -->
<a href="mailto:info@ejemplo.com">Contactar</a>

<!-- Enlace de descarga -->
<a href="documento.pdf" download>Descargar PDF</a>

4) Imágenes y figuras

Incluye siempre alt para accesibilidad.
<!-- Imagen básica -->
<img src="imagen.jpg" alt="Descripción de la imagen" width="400" height="300">

<!-- Imagen con figura y leyenda -->
<figure>
  <img src="foto.jpg" alt="Foto descriptiva">
  <figcaption>Descripción o pie de foto.</figcaption>
</figure>

<!-- Responsive (alta densidad) -->
<img src="imagen-pequena.jpg" srcset="imagen-grande.jpg 2x" alt="Descripción">

5) Vídeo con controles

La opción controls muestra los botones de reproducción, volumen, etc.
Atributos útiles: controls, loop, autoplay, poster="preview.jpg", width, height.
<!-- Vídeo básico con controles -->
<video src="videos/mi_video.mp4" width="640" height="360" controls>
  Tu navegador no soporta el elemento de video.
</video>

<!-- Versión completa -->
<video src="videos/mi_video.mp4" width="600" height="400" controls loop poster="videos/preview.jpg">
  Tu navegador no soporta el video.
</video>

6) Listas

<!-- Lista ordenada -->
<ol>
  <li>Paso 1</li>
  <li>Paso 2</li>
  <li>Paso 3</li>
</ol>

<!-- Lista desordenada -->
<ul>
  <li>Elemento A</li>
  <li>Elemento B</li>
  <li>Elemento C</li>
</ul>

7) Tablas

Usa <thead> para cabecera y <tbody> para el cuerpo.
<table border="1">
  <thead>
    <tr>
      <th>Nombre</th>
      <th>Edad</th>
      <th>Ciudad</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Juan</td>
      <td>25</td>
      <td>Madrid</td>
    </tr>
    <tr>
      <td>Lucía</td>
      <td>22</td>
      <td>Sevilla</td>
    </tr>
  </tbody>
</table>

8) Formularios

Métodos comunes: GET y POST. Usa required para validación básica.
<form action="procesar.php" method="post">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" required>

  <label for="correo">Correo:</label>
  <input type="email" id="correo" name="correo">

  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje" name="mensaje"></textarea>

  <!-- Select y opciones -->
  <label for="ciudad">Ciudad:</label>
  <select id="ciudad" name="ciudad">
    <option value="mad">Madrid</option>
    <option value="sev">Sevilla</option>
    <option value="bcn">Barcelona</option>
  </select>

  <!-- Checkbox -->
  <label>
    <input type="checkbox" name="acepto" required> Acepto las condiciones
  </label>

  <!-- Botón de envío -->
  <input type="submit" value="Enviar">
</form>
